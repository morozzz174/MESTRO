# Скриншоты для RuStore

## Требования RuStore

### Количество
- Минимум: **2 скриншота**
- Рекомендуется: **5-10 скриншотов**

### Размеры
| Тип | Размер (пиксели) | Формат |
|-----|------------------|--------|
| Телефон (Portrait) | 1080x1920 | PNG/JPG |
| Телефон (Landscape) | 1920x1080 | PNG/JPG |
| Планшет (Portrait) | 2560x1600 | PNG/JPG |

### Рекомендации
1. Покажите основные функции приложения
2. Используйте реальные данные (не тестовые)
3. Избегайте конфиденциальной информации
4. Не обрезайте системные элементы интерфейса

## Что нужно сфотографировать

### Обязательно:
1. **Главный экран / Дашборд** — покажите основной интерфейс
2. **Чек-лист** — пример заполнения
3. **Календарь замеров** — функция планирования
4. **План помещения** — 2D/2.5D режим
5. **Генерация PDF** — итоговый документ

### Желательно:
6. **Профиль пользователя** — экран настроек
7. **Клиенты** — список карточек
8. **Голосовой ввод** — диалог ввода

## Как сделать скриншоты

### На устройстве:
1. Откройте нужный экран
2. Нажмите: **Power + Volume Down** одновременно
3. Скриншот сохранится в Галерею

### Через ADB (компьютер):
```bash
# Подключите устройство и включите USB-отладку
adb devices
adb shell screencap -p /sdcard/screenshot.png
adb pull /sdcard/screenshot.png ./rustore_screenshots/
```

### Через эмулятор Android Studio:
1. Запустите эмулятор
2. Откройте приложение
3. Используйте инструмент захвата в Android Studio
