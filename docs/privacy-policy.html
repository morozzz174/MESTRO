3.1. Персональные данные обрабатываются исключительно для следующих целей:

  • Фотофиксация — привязка фотографий к объектам замеров
  • Геолокация — автоматическая привязка GPS-координат к фото
  • Уведомления — напоминания о предстоящих замерах
  • Поддержка — обработка обращений в службу поддержки
  • Аналитика — обезличенная статистика использования для улучшения Приложения
  • Обработка платежей — подтверждение оплаты премиум-подписки (через ЮKassa)

5. ХРАНЕНИЕ И ЗАЩИТА ДАННЫХ

5.1. Локальное хранение. Все персональные данные Пользователя хранятся исключительно на устройстве Пользователя в локальной базе данных SQLite. Данные не передаются на серверы Оператора.

5.2. Исключения: Данные передаются третьим лицам только в следующих случаях:

  • LoginBot (номер телефона) — для верификации при регистрации
  • ЮKassa (данные для оплаты) — для обработки платежей премиум-подписки
  • По запросу уполномоченных органов в случаях, предусмотренных законодательством РФ

5.3. Меры защиты:

  • Данные хранятся в локальной базе данных на устройстве
  • Доступ к данным возможен только через Приложение
  • Резервные копии создаются по инициативе Пользователя и хранятся на его устройстве
  • Платёжные данные обрабатываются через защищённое соединение (HTTPS) платёжной системы ЮKassa

5.4. Оператор не несёт ответственности за сохранность данных в случае:

  • Потери или повреждения устройства Пользователя
  • Удаления Приложения Пользователем
  • Сброса настроек устройства до заводских

5.5. Рекомендация: Регулярно создавайте резервные копии базы данных через функцию в Приложении.
