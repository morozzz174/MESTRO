# Политика конфиденциальности MESTRO

## Дата вступления в силу: 01.04.2024

## 1. Общие положения

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных пользователей приложения **MESTRO** (далее — «Приложение»).

Используя Приложение, вы соглашаетесь с условиями настоящей Политики.

## 2. Какие данные мы собираем

### 2.1. Данные, которые вы предоставляете нам напрямую:

- **Номер телефона** — для авторизации через SMS-код
- **ФИО** — для идентификации в заявках
- **Данные клиентов** — имена, адреса, телефоны ваших заказчиков
- **Фотографии** — снимки объектов замера с геотегами
- **Данные чек-листов** — результаты замеров
- **Информация об оплате** — данные о подписке (обрабатываются через RuStore)

### 2.2. Данные, собираемые автоматически:

- **Геолокация** — координаты места проведения замера (при наличии разрешения)
- **Данные об устройстве** — модель, версия Android, уникальный идентификатор
- **Статистика использования** — информация о работе функций Приложения

## 3. Как мы используем данные

Собранные данные используются исключительно для:

1. **Авторизации** — верификация через SMS-код
2. **Создания заявок** — хранение данных о замерах
3. **Фотофиксации** — сохранение и отображение снимков
4. **Расчёта стоимости** — автоматический расчёт по прайс-листам
5. **Генерации документов** — создание PDF-коммерческих предложений
6. **Уведомлений** — напоминания о запланированных замерах
7. **Голосового ввода** — распознавание речи на устройстве (без отправки на сервер)

## 4. Локальное хранение данных

**Все данные хранятся локально на вашем устройстве** в зашифрованной базе данных SQLite.

Приложение **НЕ передаёт** ваши данные или данные ваших клиентов на внешние серверы, за исключением:

- Оплаты подписки через RuStore (платёжный сервис)
- Голосового ввода (при использовании системного распознавания речи)

## 5. Права пользователя

Вы имеете право:

- **Получить доступ** к своим данным
- **Удалить данные** — удалив Приложение или через функцию экспорта/удаления
- **Отозвать согласие** — удалив Приложение
- **Экспортировать данные** — через функцию резервного копирования в Приложении

## 6. Защита данных

Мы применяем следующие меры защиты:

- Шифрование локальной базы данных
- Защита файлов приложения от несанкционированного доступа
- Минимальный набор запрашиваемых разрешений
- Отсутствие передачи данных третьим лицам

## 7. Разрешения приложения

Приложение запрашивает следующие разрешения:

| Разрешение | Цель |
|-----------|------|
| Камера | Фотофиксация объектов |
| Геолокация | Геотегирование фотографий |
| Микрофон | Голосовой ввод замеров |
| Телефон | Звонок клиенту |
| Хранилище | Сохранение фотографий и резервных копий |
| Уведомления | Напоминания о замерах |
| Bluetooth | Подключение гарнитуры для голосового ввода |

## 8. Сторонние сервисы

### 8.1. RuStore (оплата подписки)

При покупке Premium-подписки через встроенные покупки:
- Платёжные данные обрабатываются RuStore
- Мы получаем только подтверждение оплаты
- Условия: [Правила RuStore](https://rustore.ru/developers/terms)

### 8.2. uCaller (авторизация)

Для SMS-авторизации используется сервис uCaller:
- Ссылка: [uCaller.ru](https://ucaller.ru/)
- Обрабатывает только код верификации
- Данные не хранятся

## 9. Хранение и удаление данных

### 9.1. Срок хранения

Данные хранятся:
- На устройстве — бессрочно, до удаления Приложения
- В резервных копиях — бессрочно, по усмотрению пользователя

### 9.2. Удаление

Для удаления всех данных:
1. Удалите Приложение с устройства
2. Удалите созданные вами резервные копии

При удалении Приложения все локальные данные удаляются автоматически.

## 10. Изменения в Политике

Мы можем обновить настоящую Политику. Об изменениях будет сообщено:
- Через уведомление в Приложении
- Обновлением даты вступления в силу

Продолжая использовать Приложение после изменений, вы соглашаетесь с обновлённой Политикой.
