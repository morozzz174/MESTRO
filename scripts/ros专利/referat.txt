1. НАЗНАЧЕНИЕ ПРОГРАММЫ

MESTRO — мобильное приложение для выездных специалистов-замерщиков, работающих в сфере строительства, ремонта и отделки. Программа автоматизирует полный цикл работы мастера: от создания заявки на замер до формирования коммерческого предложения с расчётом стоимости работ.

Приложение предназначено для использования на мобильных устройствах под управлением операционной системы Android и iOS, работает в офлайн-режиме (без требования постоянного подключения к сети Интернет после первичной регистрации).

2. ОБЛАСТЬ ПРИМЕННЕНИЯ

Программа применяется в следующих отраслях:
• Остекление и оконные системы
• Дверные конструкции
• Климатическое оборудование (кондиционеры, вентиляция)
• Кухонные гарнитуры и мебель
• Плиточные и отделочные работы
• Инженерные коммуникации (отопление, водоснабжение, канализация)
• Электромонтажные работы
• Строительство и проектирование

Целевые пользователи: мастера-замерщики, монтажники, строительные бригады, дизайнеры и проектировщики.

3. ФУНКЦИОНАЛЬНЫЕ ВОЗМОЖНОСТИ

3.1. Система заявок (Orders)
• Создание, редактирование и удаление заявок на замер
• Привязка клиента (ФИО, телефон, адрес объекта)
• Назначение даты и времени выезда
• Управление статусами заявок (Новая, В работе, Завершена, Отменена)
• Привязка типа работ (8 специализаций)

3.2. Динамические чек-листы
• Загрузка конфигураций чек-листов из структурированных JSON-файлов
• Условная видимость полей (поле появляется/скрывается в зависимости от предыдущих ответов)
• Поддержка типов полей: текст, число, выбор из списка, булево значение, дата, фото
• Валидация обязательных полей перед завершением замера

3.3. Фотофиксация с аннотациями
• Съёмка с камеры устройства или выбор из галереи
• Нанесение аннотаций поверх фото: стрелки, круги, текст
• Автоматическая привязка GPS-координат к каждому фото
• Хранение аннотированной и оригинальной версии фотографии

3.4. Калькулятор стоимости
• Автоматический расчёт стоимости на основе данных чек-листа и прайс-листа
• Редактируемые цены (пользовательские цены сохраняются локально)
• Формульный расчёт количества из полей чек-листа (например: (ширина/1000) × (высота/1000))
• Безопасный парсер математических выражений для вычисления формул

3.5. Генератор планов помещений (Floor Plan)
• Rule Engine — автоматическая генерация плана помещения из замеров по правилам СНиП
• AI-оптимизация через TensorFlow Lite модель (on-device, 66 КБ)
• Интерактивный редактор: drag & drop комнат, изменение размеров, добавление/удаление помещений
• 10 типов комнат: кухня, гостиная, спальня, детская, ванная, туалет, кабинет, кладовая, коридор, балкон
• Валидация в реальном времени: контроль площадей, пересечений, СНиП-норм
• Undo/Redo — стек из 50 состояний
• Экспорт плана помещения в формат PDF

3.6. Голосовой ввод
• Распознавание речи через системный API
• Автоматическое извлечение структурированных данных (размеры, типы, количества) из текста через регулярные выражения
• Распознавание на русском языке
• Заполнение полей чек-листа надиктовкой

3.7. Календарь замеров
• Визуальное представление календаря (месяц / 2 недели / неделя)
• Отображение замеров по дням с цветными маркерами статусов
• Фильтры: прошедшие, сегодняшние, будущие замеры
• Планирование push-уведомлений

3.8. Система уведомлений
• Локальные push-уведомления
• Напоминания мастеру за 1 час и 30 минут до замера
• Планирование уведомлений в фоновом режиме

3.9. Прайс-лист
• Загрузка дефолтных цен из JSON-файлов (8 специализаций)
• Редактирование цен через пользовательский интерфейс с сохранением в базе данных
• Добавление и удаление позиций
• Сброс к дефолтным ценам

3.10. Статистика и аналитика
• Заявки по статусам с процентным соотношением
• Финансовая сводка: общая сумма, оплачено, задолженность клиентов
• Разбивка по типам работ
• Топ-5 клиентов по выручке
• Статистика платежей по месяцам

3.11. Управление платежами
• Фиксация оплат по каждой заявке
• Подсчёт оплаченной суммы и задолженности

3.12. База клиентов
• Хранение информации о клиентах (имя, телефон)
• История заявок по клиенту

3.13. Экспорт данных
• Экспорт всех заявок в формат Excel (.xlsx)
• Резервное копирование базы данных (SQLite)

3.14. Генерация PDF-документов
• Коммерческие предложения с поддержкой кириллицы
• Включение: данные чек-листа, фотографии, план помещения, стоимость
• Совместное использование PDF через системный API

4. ТЕХНОЛОГИЧЕСКИЙ СТЕК

Программа разработана на кроссплатформенном фреймворке Flutter (язык Dart) с использованием следующих технологий:

• Архитектура: паттерн BLoC (Business Logic Component)
• Локальная база данных: SQLite (sqflite)
• Голосовой ввод: speech_to_text
• Геолокация: geolocator
• Генерация PDF: pdf + printing
• Экспорт Excel: excel
• ИИ-модель: TensorFlow Lite (tflite_flutter)
• Уведомления: flutter_local_notifications
• Календарь: table_calendar

5. ЯЗЫКИ ИНТЕРФЕЙСА

Русский язык.

6. ТРЕБОВАНИЯ К СИСТЕМЕ

• Операционная система: Android 8.0+ или iOS 12.0+
• Свободное место на устройстве: не менее 200 МБ
• Доступ к камере, микрофону, геолокации
• Подключение к сети Интернет требуется только при первичной регистрации

7. УНИКАЛЬНЫЕ ОСОБЕННОСТИ

1. AI-генератор планов помещений — on-device модель TensorFlow Lite для автоматической оптимизации планировок
2. Rule Engine — автоматическая генерация плана по размерам из чек-листа с учётом строительных норм (СНиП)
3. Голосовой ввод — распознавание речи с автоматическим извлечением структурированных данных через регулярные выражения
4. Динамические чек-листы — условная видимость полей в зависимости от контекста замера
5. Формульный расчёт — каждая позиция прайс-листа имеет математическую формулу для автоматического расчёта количества
6. Полный офлайн-режим — не требует сетевого подключения после регистрации
