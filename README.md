# MESTRO — Мастер, Единый Стандарт Точности Расчёта Объекта

> *«Мастер, единый стандарт точности расчёта объекта»*

Мобильное приложение для выездных специалистов (замерщики, монтажники, строительные бригады). Автоматизирует запись клиентов, заполнение чек-листов, фотофиксацию с аннотациями, расчёт стоимости и генерацию строительных чертежей.

**🆕 Новое в v1.2:** 7 строительных специализаций, офлайн AI-агент, премиум-подписка, генератор строительных чертежей!

## 🎯 Возможности

### 📋 Чек-листы и замеры
- **Динамические чек-листы** — поля появляются/скрываются в зависимости от ответов
- **Фотофиксация с аннотациями** — стрелки, круги, текст поверх фото
- **Геотеги** — координаты автоматически привязываются к каждому фото
- **Голосовой ввод** — надиктовка замеров с автозаполнением полей
- **Расчёт стоимости** — авторасчёт по редактируемым прайс-листам

### 🤖 Офлайн AI-агент (PREMIUM)
- **Анализ данных замера** — поиск пропущенных полей, аномалий, несоответствий
- **Рекомендации по типу работ** — советы по материалам, технологиям, СНиП
- **Анализ стоимости** — проверка корректности расчётов для 15 типов работ
- **Авто-отчёты** — генерация резюме замера с оценкой уверенности
- **100% офлайн** — без отправки данных на сервер

### 🏗️ Генератор планов помещений
- **Rule Engine** — автоматическая генерация плана из размеров замера с учётом СНиП
- **AI Floor Plan Generator** — строит точный план из реальных данных чек-листа
- **AI-оптимизация** — TFLite модель для оптимальной расстановки комнат
- **Интерактивный редактор** — drag & drop комнат, resize, добавление/удаление
- **Валидация в реальном времени** — проверка площадей, пересечений, размеров
- **Undo/Redo** — стек из 50 состояний
- **Экспорт PDF** — план с таблицей комнат и compliance score

### 📐 Строительные чертежи (PREMIUM)
- **Полный комплект чертежей** в PDF (A4 landscape, ГОСТ 21.501-2011)
- **Титульная страница** — проект, заказчик, состав проекта
- **План этажа** — SVG с комнатами, стенами, дверями, окнами, подписями
- **Размерные линии и оси** — таблицы размеров, осей, отметок уровней
- **Экспликация помещений** — площадь, размеры, двери, окна, SNiP, объём
- **Ведомость стен** — материал, толщина, утепление, несущие стены
- **Схема фундамента** — тип, бетон, арматура, гидроизоляция, дренаж
- **Схема кровли** — тип, материал, стропила, утепление, водосток
- **Инженерные системы** — отопление, водоснабжение, электрика, вентиляция
- **Спецификация материалов** — полный список с количествами
- **Расчёт теплопотерь** — поэлементный расчёт + рекомендация мощности котла

### 📄 Документы
- **PDF коммерческих предложений** — с кириллицей, фото, планом помещения и ценой
- **Календарь замеров** — визуальный календарь с оптимизацией маршрутов
- **Экспорт в Excel** — все заявки в .xlsx
- **Резервное копирование** — экспорт/импорт базы данных SQLite
- **Согласие на обработку ПДн** — 152-ФЗ, реквизиты ИП

### 📱 Приложение
- **Офлайн-режим** — все данные хранятся локально в SQLite
- **Авторизация по телефону** — звонок от uCaller с кодом верификации
- **Премиум-подписка** — AI-агент, строительные чертежи, расширенные отчёты
- **Локальные уведомления** — напоминания за 1 час и 30 минут до замера

## 🏗️ Генератор планов

### Как работает

```
Данные чек-листа → AI Floor Plan Generator → FloorPlan (Extended) → Редактор → PDF / Чертежи
```

1. **AI Floor Plan Generator** анализирует данные чек-листа и строит точный план с реальными размерами
2. **Rule Engine** генерирует базовый план на основе ширины/высоты
3. **AI оптимизатор** (TFLite) улучшает расстановку комнат
4. **Интерактивный редактор** позволяет двигать, изменять размеры, добавлять комнаты
5. **Валидатор СНиП** проверяет площади, пересечения, освещение в реальном времени
6. **Экспорт PDF** — план с таблицей комнат и compliance score
7. **Строительный чертёж** — полный комплект документации (PREMIUM)
