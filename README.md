# MESTRO — Мастер, Единый Стандарт Точности Расчёта Объекта

> *«Мастер, единый стандарт точности расчёта объекта»*

[![Version](https://img.shields.io/badge/version-1.2.0-blue.svg)](https://github.com/morozzz174/MESTRO)
[![Platform](https://img.shields.io/badge/platform-Android-green.svg)](https://play.google.com/store)
[![License](https://img.shields.io/badge/license-Proprietary-red.svg)](LICENSE)

Мобильное приложение для выездных специалистов (замерщики, монтажники, строительные бригады). Автоматизирует запись клиентов, заполнение чек-листов, фотофиксацию с аннотациями, расчёт стоимости и генерацию строительных чертежей.

## 🚀 Ключевые возможности

### 📱 Для замерщика за 5 минут

```
📋 Чек-лист → 📸 Фото → 💰 Расчёт → 📄 PDF → ✅ КП готово
```

- **15 специализаций** — от окон до фундаментов ИЖС
- **Голосовой ввод** — диктуйте замеры, AI заполнит все поля
- **Фотофиксация** — стрелки, круги, текст + геотеги
- **Авторасчёт** — мгновенная стоимость по прайс-листу
- **PDF за 10 секунд** — коммерческое предложение с фото

### 🤖 AI-ассистент (Premium)

| Функция | Описание |
|---------|----------|
| **SmartChecklistAnalyzer** | Анализ данных замера, поиск ошибок и пропусков |
| **AI-валидация КП** | Проверка коммерческого предложения перед отправкой |
| **Авто-заметки замерщика** | Генерация структурированных заметок |
| **Голосовой ввод** | 15 специализаций с автозаполнением чек-листа |

### 🏗️ Конструктив здания

Полный комплект данных для строительства:

- **Стены** — тип, материал, толщина, высота, утепление
- **Фундамент** — тип, бетон, арматура, гидроизоляция
- **Кровля** — тип, материал, стропила, утепление
- **Перекрытия** — тип, толщина, звукоизоляция
- **Инженерия** — отопление, водоснабжение, электрика, вентиляция

## 📋 15 типов работ

| # | Тип | JSON | Описание |

| 5 | 🔲 Плитка | `tiles.json` | Площадь, укладка, тёплый пол |

| 8 | ⚡ Электрика | `electrical.json` | Розетки, освещение, щиток |

lib/
├── main.dart                          # Точка входа, BLoC, dotenv
│

│   ├── ai_insight.dart               # AI-анализ

├── bloc/                             # State Management

│   └── checklist_bloc.dart           # Чек-листы

│   ├── registration_screen.dart      # Авторизация по телефону
│   ├── checklist_screen.dart         # Замер (AI-агент, голос)
│   └── consent_screen.dart           # Согласие на ПДн
│
├── features/                         # Модули приложения
│   ├── home/                        # Дашборд (4 вкладки)
│   ├── calendar/                    # Календарь замеров
│   ├── floor_plan/                 # Планы + 2.5D
│   │   ├── models/                 # FloorPlan, EditorState
│   │   ├── engine/                 # RuleEngine, AI, Validator
│   │   └── presentation/          # Editor, Painter, Toolbar

│   ├── price_list/                 # Прайс-листы
│   ├── voice/                      # Голосовой ввод
│   └── notifications/              # Уведомления
│
├── services/                        # Бизнес-логика
│   ├── ai_agent_service.dart       # SmartChecklistAnalyzer
│   ├── ai_premium_agent.dart       # AI-валидация, заметки
│   ├── voice_input_service.dart    # Голос → данные
│   ├── pdf_generator.dart          # PDF КП
│   ├── construction_drawing_generator.dart # Строительные чертежи
│   └── subscription_service.dart   # Premium подписка

└── utils/
    ├── cost_calculator.dart        # Расчёт стоимости

assets/

├── fonts/                          # Шрифты для PDF
