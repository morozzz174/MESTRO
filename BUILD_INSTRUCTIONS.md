# 📦 Инструкция по сборке APK для RuStore

## Шаг 1: Подготовка

```bash
# Переход в директорию проекта
cd c:\mestro_2

# Обновление зависимостей
flutter pub get

# Очистка старых артефактов
flutter clean
```

## Шаг 2: Создание .env файла

Создайте файл `c:\mestro_2\.env`:

```env
UCALLER_SERVICE_ID=ваш_service_id
UCALLER_SECRET_KEY=ваш_secret_key
```

## Шаг 3: Сборка APK v1.3.0

```bash
flutter build apk --release
```

**Файл будет здесь:**
```
c:\mestro_2\build\app\outputs\flutter-apk\app-release.apk
```

**Переименуйте в:**
```
mestro_v1.3.0.apk
```

**Время сборки:** 2-5 минут (первый раз дольше).

**Файл:** `c:\mestro_2\build\app\outputs\bundle\release\app-release.aab`

---

## 🔑 Создание release-подписи (когда будете готовы)

```bash
keytool -genkey -v -keystore c:\mestro_2\android\app\mestro-release.keystore -alias mestro-key -keyalg RSA -keysize 2048 -validity 10000
```

Запомните пароль! Без него не сможете обновлять приложение.

## 📋 Чек-лист перед загрузкой в RUstore

### APK
- [ ] APK собран (`flutter build apk --release`)
- [ ] Размер < 150 МБ
- [ ] Приложение запускается на устройстве
- [ ] Все 15 типов работ работают
- [ ] AI-агент активируется
- [ ] Floor Plan генерируется
- [ ] PDF экспортируется

### Страница приложения
- [ ] Иконка 512×512 PNG (прозрачный фон)
- [ ] Скриншоты 3-8 шт (1080×1920 или 1080×2340)
- [ ] Описание заполнено (до 4000 символов)
- [ ] Короткое описание (до 80 символов)
- [ ] Что нового (до 500 символов)
- [ ] Категория: Продуктивность / Бизнес
- [ ] Возрастной рейтинг: 6+

### Юридическое
- [ ] Политика конфиденциальности (ссылка)
- [ ] Оферта подписки (`assets/offer.md`)
- [ ] Согласие на ПДн (в приложении)

### Аккаунт RUstore
- [ ] Зарегистрирован developer account
- [ ] ИП данные заполнены (ИНН, ОГРНИП)
- [ ] Приложение создано в консоли
- [ ] APK загружен
- [ ] Отправлено на модерацию

## 📸 Рекомендуемые скриншоты

Сделайте скриншоты на реальном устройстве:

1. **Главный экран** — дашборд с заявками
2. **Чек-лист** — замер фундамента/окна
3. **AI-анализ** — экран с инсайтами и рекомендациями
4. **Floor Plan** — план помещения
5. **Строительный чертёж** — PDF с экспликацией
6. **Премиум экран** — с тарифными планами
7. **Календарь** — замеры по датам

**Как сделать скриншот:**
- На устройстве: кнопка питания + громкость вниз
- Через Android Studio: Device Mirror → Camera icon
- Через ADB: `adb shell screencap -p /sdcard/screen.png && adb pull /sdcard/screen.png`
